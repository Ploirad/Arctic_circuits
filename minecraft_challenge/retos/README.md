## 🎯 ¿De qué van estos retos?

Bienvenido a los **10 Retos de Python para Minecraft**, diseñados específicamente para prepararte para **Eurobot 2026**.

Estos retos te enseñarán Python **desde cero**, empezando por lo más básico (mostrar mensajes) hasta crear programas completos (generadores automáticos de ciudades).

### ¿Por qué Python + Minecraft?

- **Visual y motivador**: Ves los resultados de tu código inmediatamente
- **Divertido**: Aprendes programando en un juego que te gusta
- **Práctico**: Los conceptos que aprendes son los mismos que usarás en Eurobot
- **Progresivo**: Cada reto añade nuevos conceptos sin abrumar

### Conexión con Eurobot 2026

Todos los conceptos que aprendes aquí son **directamente aplicables** a la programación del robot:
- **Coordenadas X,Y,Z** → Posicionamiento del robot en el tablero
- **Bucles** → Repetir movimientos y acciones
- **Condicionales** → Tomar decisiones según sensores
- **Funciones** → Organizar el código del robot
- **Listas** → Guardar secuencias de movimientos

## 🆘 Problemas Comunes

### "ModuleNotFoundError: No module named 'mcpi'"
**Solución:** Instala la librería con `pip install raspberryjammod`

### "Connection refused"
**Solución:**
- Asegúrate de que el servidor está arrancado
- Verifica que estás dentro del mundo de Minecraft
- El mod RaspberryJamMod debe estar instalado

### "IndentationError"
**Solución:** Python necesita espacios correctos. Usa 4 espacios para indentar (VS Code lo hace automático con Tab)

### "NameError: name 'X' is not defined"
**Solución:** Has olvidado definir una variable o función antes de usarla

### "TypeError: unsupported operand type(s)"
**Solución:** Estás mezclando tipos de datos. Usa int(), str(), float() para convertir
