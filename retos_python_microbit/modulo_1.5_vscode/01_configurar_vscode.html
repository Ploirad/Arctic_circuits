🚀 Sesión 1: Configurar Antigravity

Módulo 1.5: Programando desde Antigravity

Tu editor con IA integrada

🎯 Objetivo de esta sesión

Vamos a configurar Antigravity para programar tu micro:bit usando MicroPython. Al terminar esta sesión, tendrás todo listo para escribir y ejecutar programas en tu placa.

🤖 ¿Qué es Antigravity?

✨ Un editor de código con superpoderes

Antigravity es un editor de código creado por Google, basado en Visual Studio Code. Es como VS Code, pero con inteligencia artificial integrada que te ayuda a programar.

🎁 ¿Por qué usamos Antigravity?

  • Familiar: Si conoces VS Code, ya conoces Antigravity
  • IA integrada: Un asistente de IA te ayuda mientras programas
  • Extensiones fáciles: Usa el repositorio Open VSX, donde está la extensión de micro:bit
  • Gratis: Disponible para Windows, Mac y Linux
  • Futuro: La IA te ayudará cada vez más en tus proyectos de robótica

📥 Paso 1: Instalar los programas necesarios

1 Instalar Python 3

La extensión de micro:bit necesita Python instalado en tu ordenador.

  • Ve a: python.org/downloads
  • Descarga la última versión de Python 3
  • ¡Importante! Durante la instalación, marca la casilla "Add Python to PATH"

💡 ¿Ya tienes Python?

Puedes verificarlo abriendo una terminal y escribiendo: python --version

2 Descargar Antigravity

Abre tu navegador y ve a la web oficial de Antigravity:

🌐 Ir a antigravity.google

Haz clic en el botón de descarga. La web detectará automáticamente tu sistema operativo.

3 Instalar Antigravity

  • Windows: Ejecuta el archivo .exe descargado y sigue el asistente
  • Linux: Sigue las instrucciones de la web para tu distribución
  • Mac: Arrastra Antigravity a la carpeta Aplicaciones

💡 Es igual que VS Code

Si ya has usado Visual Studio Code antes, Antigravity te resultará muy familiar. Los menús, atajos de teclado y funcionamiento son prácticamente idénticos.

🧩 Paso 2: Instalar la extensión micro:bit Python

ℹ️ ¡Buenas noticias!

Antigravity usa el repositorio Open VSX por defecto, así que la extensión micro:bit Python de MAKinteract está disponible directamente desde el panel de extensiones.

4 Abrir el panel de extensiones

En Antigravity, tienes 3 formas de abrir el panel de extensiones:

  • Opción A - Menú: Ver → Extensiones
  • Opción B - Atajo de teclado: Ctrl + Shift + X
  • Opción C - Icono: Haz clic en el icono de cuadrados en la barra lateral izquierda

5 Buscar e instalar la extensión

En el cuadro de búsqueda del panel de extensiones:

  1. Escribe: micro:bit
  2. Busca la extensión "micro:bit Python" de makinteract
  3. Haz clic en el botón azul "Install"

💡 ¡Sin complicaciones!

A diferencia de VS Code normal, en Antigravity no necesitas configurar nada especial. La extensión aparece directamente porque usa Open VSX.

6 Esperar a que termine la instalación

La instalación tarda unos segundos. Cuando termine:

  • El botón cambiará a "Disable" o "Uninstall"
  • Puede que te pida reiniciar Antigravity - hazlo si te lo pide

🔌 Paso 3: Conectar tu micro:bit

7 Conectar el cable USB

Conecta tu micro:bit al ordenador usando el cable USB. El conector micro-USB va en la parte superior de la placa.

💡 ¿Cómo sé que está conectado?

Cuando conectas la micro:bit, debería aparecer como una unidad USB en tu ordenador (como si fuera un pendrive). En Windows aparecerá como MICROBIT (D:) o similar.

⚠️ Usa un cable de datos

Asegúrate de usar un cable USB que transmita datos, no solo de carga. Si la micro:bit no aparece como unidad, prueba con otro cable.

✅ ¡Comprobación Final!

🎉 Tu entorno está listo si...

  • ✅ Python 3 está instalado (python --version funciona en terminal)
  • ✅ Antigravity está instalado y abierto
  • ✅ La extensión "micro:bit Python" de makinteract está instalada
  • ✅ Tu micro:bit aparece como unidad USB cuando lo conectas

📌 Resumen de atajos importantes

  • Ctrl + Shift + X → Abrir panel de Extensiones
  • Ctrl + Shift + P → Abrir Paleta de Comandos
  • Ctrl + S → Guardar archivo
