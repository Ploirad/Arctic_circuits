🚀 Sesión 2: Tu Primer Programa

Módulo 1.5: Programando desde Antigravity

¡Hora de programar!

🎯 Objetivo de esta sesión

Vamos a crear tu primer programa en Python para la micro:bit, escribirlo en Antigravity, y flashearlo (cargarlo) a la placa. ¡Verás cómo tu código cobra vida!

📁 Paso 1: Abrir el proyecto de ejemplo

📂 Proyecto preparado para ti

Hemos preparado una carpeta de ejemplo con el archivo main.py ya listo. Esto te ahorrará tiempo y evitará errores comunes.

1 Abrir la carpeta del proyecto

Abre la carpeta sample_code que está dentro de modulo_1.5_vscode:

  • Menú: Archivo → Abrir carpeta...
  • Atajo: Ctrl + K, después Ctrl + O

Navega hasta la carpeta sample_code y haz clic en "Seleccionar carpeta".

⚠️ ¡Importante!

Debes abrir la carpeta que contiene main.py, no el archivo directamente. La extensión necesita un proyecto (carpeta) abierto para funcionar correctamente.

2 Verificar que el archivo main.py está presente

En el panel del explorador (izquierda), deberías ver el archivo main.py. Haz clic en él para abrirlo.

💡 ¿Por qué main.py?

La extensión de micro:bit busca específicamente un archivo llamado main.py para ejecutarlo en la placa. Este nombre es obligatorio.

👀 Paso 2: Entender el código

3 Revisar el código de ejemplo

El archivo main.py contiene este código:

# Mi primer programa para micro:bit desde Antigravity from microbit import * # Mostramos un mensaje en la pantalla LED display.scroll("Hola!") # Mostramos un corazón display.show(Image.HEART)

💡 ¿Qué hace este código?

  • from microbit import * → Importa todas las funciones de micro:bit
  • display.scroll("Hola!") → Muestra "Hola!" desplazándose por la pantalla
  • display.show(Image.HEART) → Muestra un corazón ❤️

⚡ Paso 3: Flashear a la micro:bit

4 Conectar tu micro:bit

Asegúrate de que tu micro:bit esté conectada al ordenador por USB (debería aparecer como unidad MICROBIT).

5 Flashear usando la Paleta de Comandos

Este es el método más fiable para cargar el programa en la micro:bit:

  1. Abre la Paleta de Comandos:
    • Menú: Ver → Paleta de comandos
    • Atajo: Ctrl + Shift + P
  2. Escribe: micro:bit-python: Flash sketch
  3. Selecciona el comando "micro:bit-python: Flash sketch on the micro:bit"
  4. Presiona Enter

⚠️ Usa siempre la Paleta de Comandos

Aunque existe el atajo Ctrl+F6, recomendamos usar la Paleta de Comandos (Ctrl+Shift+P) porque es más fiable y evita errores.

6 Esperar y observar

Verás lo siguiente:

  • El LED amarillo de la micro:bit parpadeará (está recibiendo el programa)
  • Antigravity mostrará un mensaje de progreso en la parte inferior
  • Cuando termine, ¡tu programa se ejecutará automáticamente!

⚠️ ¡No desconectes el cable!

Espera a que el LED deje de parpadear antes de desconectar. Si lo haces antes, podrías corromper el flasheo.

🎉 ¡Resultado!

Si todo ha ido bien, deberías ver:

  1. El texto "Hola!" desplazándose por la pantalla LED
  2. Un corazón ❤️ apareciendo al final

🔄 Para ejecutar de nuevo

Puedes reiniciar el programa pulsando el botón de RESET en la parte trasera de la micro:bit, o desconectando y volviendo a conectar el USB.

🏆 Tu Reto

¡Personaliza tu programa!

Objetivo: Modifica el programa para que muestre tu nombre y un icono diferente.

Pasos:

  1. Abre el archivo main.py
  2. Cambia "Hola!" por tu nombre
  3. Cambia Image.HEART por otro icono (prueba: Image.HAPPY, Image.SMILE, Image.DUCK)
  4. Guarda el archivo: Ctrl + S
  5. Flashea de nuevo: Ctrl + Shift + P → "Flash sketch"

Bonus:

  • Añade más líneas para mostrar varios mensajes seguidos
  • Prueba a usar display.show() en lugar de display.scroll() - ¿qué diferencia hay?

📌 Resumen de comandos importantes

  • Ctrl + K, Ctrl + O → Abrir carpeta
  • Ctrl + S → Guardar
  • Ctrl + Shift + P → Abrir Paleta de Comandos
  • Comando: micro:bit-python: Flash sketch on the micro:bit → Flashear programa
