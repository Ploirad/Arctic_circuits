💻 Sesión 3: Línea de Comandos

Módulo 1.5: Programando desde VS Code

Nivel avanzado: usa la terminal

🎯 Objetivo de esta sesión

Vamos a aprender a flashear programas a la micro:bit usando la línea de comandos (terminal). Esto es útil cuando quieres más control o automatizar el proceso.

🤔 ¿Por qué usar la línea de comandos?

  • Es más rápido una vez que lo dominas
  • Puedes automatizar tareas con scripts
  • Funciona sin necesidad de abrir VS Code
  • Es una habilidad importante para cualquier programador

📦 Paso 1: Instalar uflash

1 Abrir la terminal en VS Code

Hay varias formas de abrir la terminal integrada:

  • Menú: Ver → Terminal
  • Menú alternativo: Terminal → Nuevo terminal
  • Atajo: Ctrl + ` (la tecla encima del Tab)

Aparecerá un panel en la parte inferior de VS Code con la terminal.

2 Instalar uflash con pip

uflash es una herramienta que permite flashear programas MicroPython a la micro:bit desde la terminal.

Escribe este comando y presiona Enter:

$ pip install uflash

Verás algo como esto cuando termine:

Collecting uflash Downloading uflash-2.0.0-py3-none-any.whl Installing collected packages: uflash Successfully installed uflash-2.0.0

⚠️ ¿Error de permisos en Linux?

Si te da error, prueba: pip install --user uflash

3 Verificar la instalación

Comprueba que uflash está instalado correctamente:

$ uflash --version uflash 2.0.0

Si ves un número de versión, ¡está listo!

⚡ Paso 2: Flashear tu programa

4 Navegar a tu carpeta de proyectos

Si ya tienes abierta tu carpeta en VS Code, la terminal ya estará en el directorio correcto. Si no, navega con:

$ cd ruta/a/tu/carpeta/mis_programas_microbit

💡 Truco: ver archivos

Usa ls (Linux/Mac) o dir (Windows) para ver los archivos en la carpeta actual.

5 Flashear el programa

Asegúrate de que tu micro:bit esté conectada, y ejecuta:

$ uflash hola_mundo.py

Verás algo como:

Flashing hola_mundo.py to: /media/MICROBIT Done!

¡El LED de la micro:bit parpadeará y tu programa se ejecutará!

📚 Referencia de comandos uflash

Comandos más útiles

# Flashear un archivo específico $ uflash mi_programa.py # Flashear a una micro:bit específica (si tienes varias) $ uflash mi_programa.py /ruta/a/MICROBIT # Ver ayuda completa $ uflash --help # Ver versión $ uflash --version

💡 Consejo Pro: Historial de comandos

Usa la tecla ↑ (flecha arriba) para recuperar comandos anteriores. ¡No tienes que escribir todo otra vez!

🔧 Solución de problemas

❌ "uflash: command not found"

uflash no está en el PATH. Prueba:

  • Cierra y abre la terminal
  • Usa python -m uflash hola_mundo.py en su lugar

❌ "No micro:bit found"

La micro:bit no está conectada o no se detecta:

  • Verifica que el cable USB esté bien conectado
  • Comprueba que aparece la unidad MICROBIT
  • Prueba otro puerto USB

🏆 Tu Reto

¡Crea y flashea desde la terminal!

Objetivo: Crear un nuevo programa y flashearlo usando solo la línea de comandos.

Pasos:

  1. Crea un archivo llamado contador.py con el siguiente código:
# Contador del 1 al 5 from microbit import * for i in range(1, 6): display.show(i) sleep(1000) display.show(Image.YES)
  1. Flashéalo con: uflash contador.py
  2. ¡Observa el contador en tu micro:bit!

Bonus:

  • Modifica el contador para que cuente hacia atrás (5, 4, 3, 2, 1...)
  • Cambia el tiempo de espera entre números
