Programando micro:bit desde Antigravity

Editor con IA integrada

En este módulo aprenderás a programar tu micro:bit usando Antigravity, un editor de código de Google basado en Visual Studio Code pero con inteligencia artificial integrada. También aprenderás a flashear programas desde la línea de comandos. 🚀

📋 Antes de empezar necesitas:

  • Una micro:bit (v1 o v2)
  • Un cable USB
  • Un ordenador (Windows, Mac o Linux)
  • Conexión a internet (para descargar VS Code)
