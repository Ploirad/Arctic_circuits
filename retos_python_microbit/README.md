# 🤖 Retos de Python con micro:bit

Bienvenidos al curso de programación Python basado en el currículo de **Python Essentials** de Cisco Networking Academy, pero adaptado para que aprendas programando tu **micro:bit**.

## 🎯 ¿Qué es este proyecto?

Este repositorio contiene una serie de retos prácticos de programación diseñados para estudiantes de robótica. Cada reto está alineado con el currículo oficial de Python Essentials, pero orientado al uso del micro:bit y su ecosistema de hardware.

## 🚀 ¿Cómo funciona?

Cada módulo del curso tiene su propia carpeta con retos individuales. Cada reto:
- 📝 Explica brevemente los conceptos clave
- 🎮 Presenta un desafío práctico usando el micro:bit
- 💡 Te ayuda a aplicar los conceptos de Python en hardware real

## 🛠️ Herramientas necesarias

Para completar estos retos necesitarás:

1. **Hardware**:
   - micro:bit (v1 o v2)
   - Cable USB para conectar el micro:bit a tu ordenador

2. **Software**:
   - Navegador web moderno (Chrome, Firefox, Edge, Safari)
   - Editor online: [https://python.microbit.org/](https://python.microbit.org/)

## 📚 Estructura del Curso

El curso está dividido en módulos siguiendo el currículo de Python Essentials 1:

### Módulo 1: Introducción a Python y Programación
- Fundamentos de programación
- Tu primer programa en micro:bit
- Función `print()` y salida de datos
- Literales y tipos de datos básicos
- Operadores básicos
- Variables

### Módulo 1.5: Programando desde VS Code (Nuevo)
- Configurar Visual Studio Code para micro:bit
- Tu primer programa desde VS Code
- Flasheando desde la línea de comandos con `uflash`

### Módulo 2: Tipos de Datos, Variables y Operaciones (Próximamente)
- Entrada y salida de datos
- Operadores y expresiones
- Comentarios

### Módulo 3: Estructuras de Control (Próximamente)
- Valores booleanos
- Ejecución condicional
- Bucles
- Listas y procesamiento de listas
- Operaciones lógicas y de bits

### Módulo 4: Funciones y Estructuras de Datos (Próximamente)
- Funciones
- Tuplas y diccionarios
- Excepciones
- Procesamiento de datos

## 🎓 Metodología de Aprendizaje

1. **Lee** la breve explicación de conceptos en cada reto
2. **Experimenta** con los ejemplos proporcionados
3. **Resuelve** el reto propuesto
4. **Prueba** tu código en el micro:bit real
5. **Itera** y mejora tu solución

## 🌟 Consejos para el Éxito

- ✅ **No te saltes retos**: Cada uno construye sobre los anteriores
- ✅ **Experimenta**: Prueba diferentes soluciones
- ✅ **Juega con el hardware**: El micro:bit es para tocarlo y probarlo
- ✅ **Pide ayuda**: La programación es mejor en comunidad
- ✅ **Diviértete**: ¡Estás aprendiendo programación haciendo cosas reales!
