Sección 1.1 - Fundamentos

📚 Conceptos Clave

¿Qué es un programa?

Un programa es un conjunto de instrucciones que le das a una computadora (o en este caso, a tu micro:bit) para que realice una tarea específica. Es como una receta de cocina: paso a paso, le dices qué hacer.

¿Qué es Python?

Python es un lenguaje de programación que nos permite comunicarnos con las computadoras de forma sencilla. Es como aprender un nuevo idioma, pero en lugar de hablar con personas, hablas con máquinas.

La función display.show()

En micro:bit, usamos display.show() para mostrar información en la pantalla LED (similar a cómo en Python estándar usamos print()). Es tu forma de ver lo que el programa está haciendo.

💡 Ejemplo

Aquí está el programa más simple que puedes escribir en micro:bit:

from microbit import * # Este es tu primer programa display.show("Hola")

💡 Explicación del código:

  • from microbit import * - Importa todas las funciones de micro:bit
  • display.show("Hola") - Muestra "Hola" en la pantalla LED del micro:bit
  • Las líneas que empiezan con # son comentarios y el programa las ignora
