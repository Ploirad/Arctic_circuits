📚 Conceptos Clave

La función display.show()

display.show() es la función principal para mostrar información en el micro:bit. Puede mostrar texto, números, e incluso imágenes predefinidas. Es equivalente a la función print() de Python estándar, pero visualiza en la matriz LED 5x5.

Mostrando texto vs números

Puedes mostrar tanto texto (strings) como números. El texto va entre comillas ("" o ''), mientras que los números no necesitan comillas.

Imágenes predefinidas

El micro:bit tiene muchas imágenes incorporadas como Image.HAPPY, Image.HEART, Image.SAD, etc. Son perfectas para crear programas expresivos.

💡 Ejemplos

Ejemplo 1: Mostrar texto

from microbit import * # Mostrar un mensaje display.show("Python")

Ejemplo 2: Mostrar números

from microbit import * # Mostrar un número display.show(42)

Ejemplo 3: Mostrar imágenes

from microbit import * # Mostrar una carita feliz display.show(Image.HAPPY)

Ejemplo 4: Mostrar varias cosas con scroll

from microbit import * # El texto largo se desplaza automáticamente display.scroll("Hola, bienvenido al mundo de Python!")

💡 Diferencia entre show() y scroll()

  • show() - Muestra el contenido de forma estática (o letra por letra si es texto)
  • scroll() - Desplaza el contenido de derecha a izquierda continuamente

🎯 Tu Reto

🎨 Crea un programa de presentación

Objetivo: Crear un programa que muestre diferentes tipos de información sobre ti.

Requisitos mínimos:

  • Mostrar tu nombre usando display.scroll()
  • Mostrar tu edad usando display.show() con un número
  • Mostrar una imagen que te represente (ej: Image.HAPPY)
  • Usa sleep() para hacer pausas entre cada elemento (ej: sleep(1000) para 1 segundo)

Bonus (opcional):

  • Muestra una secuencia de imágenes (como una animación simple)
  • Combina texto y números en un solo mensaje
  • Investiga cómo usar display.clear() para limpiar la pantalla
  • Haz que el programa se repita en bucle infinito con while True:

⚠️ Importante

Si usas while True: para hacer un bucle infinito, recuerda agregar pausas con sleep() entre cada ciclo. De lo contrario, el programa será muy rápido y no verás bien lo que muestra.

🔍 Ejemplo de Solución

Aquí tienes una posible solución (¡intenta hacerla tú primero!):

from microbit import * # Programa de presentación display.scroll("Hola soy") display.scroll("Juan") sleep(500) display.scroll("Tengo") display.show(12) sleep(1000) display.show(Image.HAPPY) sleep(2000) display.clear()
