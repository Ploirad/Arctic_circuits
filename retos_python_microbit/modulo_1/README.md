# 📖 Módulo 1: Introducción a Python y Programación

¡Bienvenido al primer módulo del curso de Python con micro:bit! 🎉

## 🎯 Objetivos del Módulo

En este módulo aprenderás los fundamentos de la programación en Python mientras creas programas interactivos para tu micro:bit. Al final de este módulo serás capaz de:

- ✅ Entender qué es un programa y cómo se ejecuta
- ✅ Escribir tu primer programa en Python
- ✅ Mostrar información en la pantalla LED del micro:bit
- ✅ Trabajar con diferentes tipos de datos (literales)
- ✅ Usar operadores para hacer cálculos
- ✅ Crear y usar variables para almacenar información

## 📚 Secciones del Módulo

### [Reto 1: Tu Primer Programa](01_tu_primer_programa.html) 🚀
**Conceptos:** Introducción a la programación, primer programa, función `display.show()`

Aprende qué es un programa y crea tu primer código para micro:bit. Harás que la pantalla LED muestre tu nombre.

**Tiempo estimado:** 15-20 minutos

---

### [Reto 2: Mostrando Datos](02_mostrando_datos.html) 📊
**Conceptos:** `display.show()`, `display.scroll()`, imágenes, `sleep()`

Descubre las diferentes formas de mostrar información en tu micro:bit: texto, números, imágenes y animaciones.

**Tiempo estimado:** 20-25 minutos

---

### [Reto 3: Literales y Datos](03_literales.html) 🔢
**Conceptos:** Literales, tipos de datos (strings, integers, floats, booleans)

Comprende qué son los literales y aprende a trabajar con los diferentes tipos de datos en Python.

**Tiempo estimado:** 25-30 minutos

---

### [Reto 4: Operadores Básicos](04_operadores.html) ➕
**Conceptos:** Operadores aritméticos (+, -, *, /, //, %, **), orden de operaciones

Convierte tu micro:bit en una calculadora. Aprende a usar operadores para hacer cálculos matemáticos.

**Tiempo estimado:** 30-35 minutos

---

### [Reto 5: Variables](05_variables.html) 📦
**Conceptos:** Variables, asignación, nombres de variables, modificación de valores

Aprende a crear "cajas" para guardar información y cómo usarlas en tus programas. Crea contadores y sistemas interactivos.

**Tiempo estimado:** 35-40 minutos

---

## 🎮 Proyecto Final del Módulo

Una vez que completes todos los retos, intenta crear un **Juego Simple** que combine todo lo aprendido:

### 🎯 Especificaciones del Proyecto:
- Usa al menos 3 variables diferentes
- Incluye operaciones matemáticas
- Muestra información usando `display.show()` y `display.scroll()`
- Responde a los botones A y B
- Usa imágenes o animaciones
- Incluye un sistema de puntuación

### 💡 Ideas de proyectos:
1. **Juego de Adivinanzas**: El micro:bit "piensa" un número y tú adivinas con los botones
2. **Contador de Ejercicios**: Cuenta flexiones/abdominales usando el acelerómetro
3. **Dado Digital**: Presiona un botón para "lanzar" un dado (número aleatorio 1-6)
4. **Termómetro Interactivo**: Muestra la temperatura con diferentes emojis según el valor
5. **Cronómetro Simple**: Cuenta segundos y muestra el tiempo
