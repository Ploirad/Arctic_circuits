Sección 1.3 - Tipos de Datos

📚 Conceptos Clave

¿Qué son los literales?

Los literales son valores fijos que escribes directamente en tu código. Son datos "literales" - exactamente lo que ves es lo que obtienes. Por ejemplo, el número 5 o el texto "Hola" son literales.

Datos vs Literales

Imagina que los datos son como ingredientes en la cocina. Los literales son cuando escribes directamente "2 huevos" o "100 gramos de harina" en tu receta (código). Son valores concretos que no cambian.

🎨 Tipos de Literales en Python

📝 Strings (Texto)

Cadenas de caracteres entre comillas

"Hola micro:bit" 'Python'

🔢 Integers (Enteros)

Números sin decimales

42 -15 0

💧 Floats (Decimales)

Números con punto decimal

3.14 -0.5 2.0

✅ Booleans (Lógicos)

Verdadero o Falso

True False

💡 Ejemplos con micro:bit

Ejemplo 1: Literales de texto (strings)

from microbit import * # Estos son literales de tipo string display.scroll("Python") display.scroll('micro:bit') display.scroll("¡Hola Mundo!")

Ejemplo 2: Literales numéricos

from microbit import * # Literales enteros display.show(100) sleep(1000) # También puedes mostrar negativos display.show(-5) sleep(1000) # Y el cero display.show(0)

Ejemplo 3: Combinando diferentes tipos

from microbit import * # Mostrar una cuenta regresiva display.show(3) sleep(1000) display.show(2) sleep(1000) display.show(1) sleep(1000) display.scroll("GO!")

💡 Nota Importante

Cuando escribes "5" (con comillas) es un string (texto), pero cuando escribes 5 (sin comillas) es un número. ¡Son cosas diferentes! Los números puedes usarlos para hacer matemáticas, el texto no.

🎯 Tu Reto

🎪 ¡Crea un programa de información!

Objetivo: Crear un programa que muestre diferentes tipos de literales.

Requisitos:

  • Muestra tu nombre (literal string)
  • Muestra tu edad (literal numérico)
  • Muestra cuántas mascotas tienes (literal numérico, puede ser 0)
  • Muestra tu comida favorita (literal string)
  • Usa sleep() entre cada dato para que se vea bien

Bonus (opcional):

  • Crea una calculadora simple que muestre el resultado de operaciones como 10 + 5
  • Muestra tu altura en centímetros (usa un decimal, por ejemplo 152.5)
  • Crea una secuencia que combine strings, números e imágenes
  • Intenta mostrar la temperatura actual usando temperature()

🔍 Ejemplo de Solución

Aquí tienes un ejemplo (¡intenta el tuyo primero!):

from microbit import * # Mi información personal usando literales display.scroll("Nombre:") display.scroll("Ana") sleep(1000) display.scroll("Edad:") display.show(13) sleep(1500) display.scroll("Mascotas:") display.show(2) sleep(1500) display.scroll("Comida:") display.scroll("Pizza") # Bonus: mostrar la temperatura display.scroll("Temp:") display.show(temperature())
