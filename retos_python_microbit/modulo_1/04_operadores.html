Sección 1.4 - Operaciones Matemáticas

📚 Conceptos Clave

¿Qué son los operadores?

Los operadores son símbolos que le dicen a Python qué hacer con los datos. Son como las señales matemáticas que conoces (+, -, ×, ÷), pero en programación. ¡Tu micro:bit puede hacer cálculos matemáticos!

Python como calculadora

Python puede funcionar como una calculadora super poderosa. Puede sumar, restar, multiplicar, dividir y mucho más. Estos cálculos son útiles para controlar robots, sensores, y crear programas interactivos.

🧮 Operadores Aritméticos

Operador Nombre Ejemplo Resultado
+ Suma 10 + 5 15
- Resta 10 - 5 5
* Multiplicación 10 * 5 50
/ División 10 / 5 2.0
// División entera 10 // 3 3
% Módulo (residuo) 10 % 3 1
** Potencia 2 ** 3 8

💡 Diferencia importante

/ siempre da un resultado decimal (float), mientras que // descarta los decimales y solo te da la parte entera.

💡 Ejemplos con micro:bit

Ejemplo 1: Suma simple

from microbit import * # Calcular la suma de dos números display.scroll("5 + 3 =") display.show(5 + 3) # Mostrará: 8

Ejemplo 2: Diferentes operaciones

from microbit import * # Multiplicación display.scroll("10 x 3") display.show(10 * 3) sleep(2000) # División entera display.scroll("20 / 6") display.show(20 // 6) # Mostrará: 3 sleep(2000) # Potencia display.scroll("2^4") display.show(2 ** 4) # Mostrará: 16

Ejemplo 3: Usando sensores

from microbit import * # Convertir temperatura de Celsius a Fahrenheit # Fórmula: (C * 9/5) + 32 celsius = temperature() fahrenheit = (celsius * 9 // 5) + 32 display.scroll("Temp C:") display.show(celsius) sleep(1500) display.scroll("Temp F:") display.show(fahrenheit)

⚠️ Orden de Operaciones

Python sigue las reglas matemáticas que conoces: primero potencias, luego multiplicaciones y divisiones, y finalmente sumas y restas. Puedes usar paréntesis () para cambiar el orden.

🎯 Tu Reto

🧮 ¡Crea una calculadora micro:bit!

Objetivo: Crear un programa que realice y muestre diferentes cálculos matemáticos.

Parte 1 - Requisitos básicos:

  • Calcula cuántas horas hay en una semana (7 días × 24 horas)
  • Calcula cuántos segundos hay en un minuto × cuántos minutos hay en una hora
  • Si tienes 100 caramelos y los repartes entre 7 amigos, ¿cuántos le tocan a cada uno? (división entera)
  • ¿Cuántos caramelos sobran? (usa el operador módulo %)

Parte 2 - Bonus con sensores:

  • Lee el nivel de luz con display.read_light_level() y calcula su porcentaje (el valor máximo es 255)
  • Calcula el doble de la temperatura actual
  • Crea un contador que sume 1 cada vez que presiones el botón A

Parte 3 - Desafío extra:

  • Crea un programa que calcule tu edad en días (edad en años × 365)
  • Calcula cuántos años tiene tu mascota en "años humanos" (edad × 7 para perros)
  • Experimenta con potencias: calcula 2^1, 2^2, 2^3, 2^4, etc.

🔍 Ejemplo de Solución

Un ejemplo para la Parte 1:

from microbit import * # Calculadora micro:bit # Horas en una semana display.scroll("Horas/semana:") display.show(7 * 24) sleep(2000) # Segundos en una hora display.scroll("Seg/hora:") display.show(60 * 60) sleep(2000) # Repartir caramelos display.scroll("100/7=") display.show(100 // 7) sleep(2000) display.scroll("Sobran:") display.show(100 % 7)

🎓 Para pensar...

El operador módulo (%) es muy útil en robótica. Por ejemplo, si quieres que algo se repita cada 10 ciclos, puedes usar contador % 10 == 0. También sirve para saber si un número es par: numero % 2 == 0.
