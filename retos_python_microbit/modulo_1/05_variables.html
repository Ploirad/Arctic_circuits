📚 Conceptos Clave

¿Qué es una variable?

Una variable es como una caja con una etiqueta donde guardas información. Puedes poner cosas dentro, sacarlas, cambiarlas, y usar lo que hay dentro cuando lo necesites. La etiqueta es el nombre de la variable, y lo que guardas dentro es su valor.

🎁 Analogía del mundo real

Imagina que tienes cajas en tu habitación. Una caja dice "juguetes", otra dice "libros", y otra dice "ropa". Puedes poner cosas en cada caja, cambiar lo que hay dentro, o sacar cosas cuando las necesites. Las variables funcionan exactamente igual, pero en tu programa.

Creando variables

En Python, crear una variable es muy fácil. Solo escribes un nombre, el símbolo =, y el valor que quieres guardar. A esto se le llama "asignar un valor a una variable".

📝 Reglas para Nombres de Variables

✅ Reglas obligatorias:

  • Solo pueden contener letras, números y guiones bajos (_)
  • Deben empezar con una letra o guión bajo (no con un número)
  • No pueden tener espacios (usa guiones bajos en su lugar)
  • No pueden ser palabras reservadas de Python (como if, for, while)
  • Python diferencia mayúsculas y minúsculas (edad ≠ Edad)

✅ Buenos nombres:

nombre edad_usuario temperatura_max contador1 _privado

❌ Nombres incorrectos:

2edad mi nombre for class temperatura-max

💡 Ejemplos con micro:bit

Ejemplo 1: Crear y usar variables

from microbit import * # Crear una variable nombre = "micro:bit" edad = 5 # Usar las variables display.scroll(nombre) display.show(edad)

Ejemplo 2: Variables con sensores

from microbit import * # Guardar lecturas de sensores en variables temp = temperature() luz = display.read_light_level() display.scroll("Temp:") display.show(temp) sleep(1500) display.scroll("Luz:") display.show(luz)

Ejemplo 3: Modificar variables

from microbit import * # Crear un contador contador = 0 while True: if button_a.was_pressed(): contador = contador + 1 display.show(contador) if button_b.was_pressed(): contador = 0 display.show(contador) sleep(100)

Ejemplo 4: Variables en cálculos

from microbit import * # Crear variables con datos base = 10 altura = 5 # Calcular área del triángulo area = (base * altura) // 2 display.scroll("Area:") display.show(area)

💡 Consejo importante

Usa nombres descriptivos para tus variables. En lugar de x o a, usa nombres como temperatura, contador_clicks, o velocidad_motor. ¡Tu código será mucho más fácil de entender!

🎮 ¡Crea un sistema de puntuación!

Objetivo: Crear un programa que use variables para llevar el control de diferentes valores.

Parte 1 - Información personal:

  • Crea variables para: tu nombre, edad, ciudad, y hobby favorito
  • Muestra toda la información en secuencia en el micro:bit
  • Usa sleep() entre cada dato para que se vea bien

Parte 2 - Contador interactivo:

  • Crea una variable puntos que empiece en 0
  • Botón A: suma 1 punto
  • Botón B: resta 1 punto (pero no puede ser negativo)
  • Botones A+B juntos: resetea a 0
  • Muestra siempre el puntaje actual en la pantalla

Parte 3 - Estadísticas:

  • Crea variables para almacenar múltiples lecturas de temperatura
  • Calcula y muestra: la suma, el promedio, o el valor máximo/mínimo
  • Usa variables para hacer seguimiento de cuántas veces se ha presionado cada botón

Bonus - Desafío avanzado:

  • Crea un "health bar" (barra de vida) que comience en 100
  • Botón A: recibe daño (-10 vida)
  • Botón B: se cura (+5 vida, máximo 100)
  • Cuando la vida llega a 0, muestra un Image.SKULL
  • Cuando la vida es mayor a 80, muestra Image.HAPPY

🔍 Ejemplo de Solución

Solución para la Parte 2 (contador interactivo):

from microbit import * # Crear la variable puntos puntos = 0 # Mostrar puntos iniciales display.show(puntos) # Bucle principal while True: # Si presionan A y B juntos if button_a.is_pressed() and button_b.is_pressed(): puntos = 0 display.show(puntos) sleep(500) # Si presionan solo A elif button_a.was_pressed(): puntos = puntos + 1 display.show(puntos) # Si presionan solo B elif button_b.was_pressed(): if puntos > 0: puntos = puntos - 1 display.show(puntos) sleep(100)

🚀 Atajo útil

En lugar de escribir puntos = puntos + 1, puedes usar el atajo puntos += 1. Funciona igual pero es más corto. También existen -=, *=, y /=.
