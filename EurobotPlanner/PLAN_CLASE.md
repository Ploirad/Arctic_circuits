# 📚 Plan de Clase: Organización del Equipo Eurobot 2026

**Duración:** 90-120 minutos
**Objetivo:** Organizar al equipo de 8 personas y asignar tareas para Eurobot 2026

---

## 🎯 Objetivos

1. Entender las 5 acciones principales del robot
2. Dividir el trabajo en tareas manejables
3. Asignar responsabilidades entre 8 miembros
4. Establecer prioridades y dependencias
5. Crear sistema de seguimiento

---

## ⏱️ Estructura de la Clase (90 min)

### PARTE 1: Introducción (15 min)

**1.1. Presentación del tema (5 min)**
- Mostrar video/imagen del campo Eurobot 2026
- Contexto: "Winter is Coming" - ardillas y bellotas

**1.2. Resumen de reglas (10 min)**

Las **5 acciones principales:**

| Acción | Puntos | Prioridad |
|--------|--------|-----------|
| 🎯 Recolectar cajas de bellotas | ~150 pts | ⭐⭐⭐ |
| 🐿️ SIMA vaciar granero | 28 pts | ⭐⭐ |
| 🌡️ Mover cursor termómetro | 10 pts | ⭐ |
| 🏠 Regresar al nido | 10 pts | ⭐⭐ |
| 🍽️ SIMAs a despensas | 55 pts | ⭐⭐ |

**Puntos clave:**
- Partido de 100 segundos
- Acción principal = mayoría de puntos
- 1 robot grande + hasta 7 pequeños (SIMAs)

---

### PARTE 2: Análisis (20 min)

**2.1. Brainstorming (10 min)**

Preguntas al equipo (anotar en pizarra):

1. "¿Qué necesitamos para mover el robot?"
   → *Motores, ruedas, chasis, controlador*

2. "¿Qué necesitamos para recoger cajas?"
   → *Pinza, brazo, sensores, almacenamiento*

3. "¿Cómo sabe el robot dónde está?"
   → *Encoders, sensores, programación*

**Categorías:**
- Mecánica
- Electrónica
- Programación
- Otros

**2.2. Mostrar la herramienta (10 min)**

Abrir `eurobot_2026_planner.html` en proyector.

**Tour rápido:**
1. Estadísticas (arriba)
2. Las 6 FASES:
   - FASE 1: Diseño y Mecánica
   - FASE 2: Manipulación
   - FASE 3: Electrónica
   - FASE 4: Programación
   - FASE 5: Pruebas
   - FASE 6: Homologación

3. Cómo asignar tareas
4. Cómo marcar progreso
5. Sección de equipo

---

### PARTE 3: Organización (30 min)

**3.1. Definir roles (10 min)**

Roles sugeridos para 8 personas:

| Rol | Responsabilidades | Personas |
|-----|-------------------|----------|
| 👷 Jefe de Mecánica | Chasis, motores, pinzas | 1-2 |
| ⚡ Jefe de Electrónica | Sensores, controladores | 1-2 |
| 💻 Jefe de Programación | Código, navegación | 1-2 |
| 🎨 Diseño CAD | Piezas 3D | 1 |
| 🧪 Testing | Pruebas, calibración | 1-2 |
| 📋 Documentación | Fotos, informes | 1 |

**Preguntar:**
- "¿Quién tiene experiencia en diseño 3D?"
- "¿Quién programa mejor?"
- "¿Quién es bueno construyendo?"

**3.2. Asignar tareas (20 min)**

**Usar la herramienta en tiempo real:**

1. Proyectar pantalla
2. Ir fase por fase
3. Leer cada tarea en voz alta
4. Preguntar: "¿Quién puede encargarse?"
5. Asignar en la herramienta
6. Verificar balance al final

**Ejemplo:**
```
Profesor: "Tarea: Diseñar chasis del robot. ¿Quién puede?"
Estudiante 1: "Yo puedo, pero necesito ayuda"
Profesor: [Asigna en herramienta] "Perfecto, Estudiante 2 te ayuda"
```

---

### PARTE 4: Estrategia (15 min)

**4.1. Decidir estrategia (10 min)**

**Estrategia MÍNIMA viable:**
```
✅ Llenar nido con 6 cajas (12 pts)
✅ Llenar 3 despensas (27 pts)
✅ Regresar al nido (10 pts)
= ~50 puntos GARANTIZADOS
```

**Estrategia COMPLETA:**
```
✅ Llenar 4-5 despensas (40-50 pts)
✅ Primas de despensas (10 pts)
✅ Llenar nido (12 pts)
✅ SIMA granero (28 pts)
✅ Termómetro (10 pts)
✅ Regresar (10 pts)
= ~100 puntos
```

**Votar:** ¿Qué estrategia seguimos?

**4.2. Priorizar tareas (5 min)**

**CRÍTICAS** (sin esto no hay robot):
- Chasis y tracción
- Controlador y batería
- Navegación básica
- Recolección de cajas
- Rutina llenar nido/despensas

**IMPORTANTES** (mejoran mucho):
- Sensores de detección
- SIMA granero
- Regresar al nido
- Campo de pruebas

**OPCIONALES** (si hay tiempo):
- Termómetro
- SIMAs extra
- Color detection

---

### PARTE 5: Plan de Trabajo (10 min)

**5.1. Cronograma (5 min)**

```
SEMANAS 1-4: Diseño + Construcción
SEMANAS 5-8: Electrónica + Programación
SEMANAS 9-12: Pruebas intensivas
SEMANAS 13-14: Homologación
SEMANA 15: COMPETICIÓN 🏆
```

**5.2. Sistema de seguimiento (5 min)**

1. **Reunión semanal** (30-45 min)
   - Cada uno reporta progreso
   - Actualizar herramienta
   - Identificar bloqueos

2. **Uso de herramienta:**
   - Empezar tarea → ▶️ "En progreso"
   - Terminar → ✅ Checkbox
   - Guardar progreso regularmente

3. **Responsabilidad:**
   - Documentación toma capturas semanales
   - Exportar JSON como backup

---

### PARTE 6: Primeros Pasos (10 min)

**6.1. Tareas inmediatas (5 min)**

"¿Qué hacemos YA esta semana?"

**Sugerencias:**
- [ ] Conseguir medidas del campo (300x200cm)
- [ ] Buscar piezas Jenga (simular cajas)
- [ ] Instalar software (Arduino IDE, FreeCAD)
- [ ] Lista de materiales para comprar
- [ ] Investigar robots años anteriores
- [ ] Empezar diseño CAD chasis

**En herramienta:**
- Marcar 2-3 tareas FASE 1 como "▶️ En progreso"

**6.2. Motivación y cierre (5 min)**

```
"Puede parecer mucho, pero está todo organizado.
Si cada uno hace sus tareas y nos ayudamos,
vamos a llegar bien preparados.

Lo importante:
✅ Comunicación: pregunta si te atascas
✅ Constancia: poco cada semana > todo al final
✅ Trabajo en equipo: nadie hace esto solo
✅ Aprender: equivocarse es parte del proceso

¿Preguntas?"
```

## 📝 Checklist Post-Clase

- [ ] Hay 2-3 tareas "en progreso" para empezar

- [ ] Fecha próxima reunión (en 1 semana)
